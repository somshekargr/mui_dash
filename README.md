# VideoSrch

## Tech

- React v18.2.0
- Node v16.17.0
- MUI v5.14.9
- react-router-dom v6.16.0

## Installation

Dillinger requires [Node.js](https://nodejs.org/) v16 to run.

Install the dependencies and devDependencies and start the server.

Open your favorite Terminal and run these commands.


## UI
First Tab:
```
cd frontend
npm install
npm start
```
```sh
127.0.0.1:3000
```
## API
Second Tab:
```
cd backend
npm install
npm start
```


```sh
127.0.0.1:8080
```
